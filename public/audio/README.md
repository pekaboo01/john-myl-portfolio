# Audio Files

Place your audio file here for the Love Page.

## Required File

- **File name**: `love-grows.mp3`
- **Song**: "Love grows" by Paravi
- **Location**: `public/audio/love-grows.mp3`

The audio will automatically play and loop when the user successfully authenticates on the Love Page.

## Supported Formats

The audio player supports common audio formats:
- MP3 (recommended)
- OGG
- WAV
- M4A

Make sure the file is named exactly `love-grows.mp3` for it to work properly.
